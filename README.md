# Poddy
A simple podcast player in the jungle of overcomplicated podcast players.

*Note!* Work in progress, missing basic features.

## Planned features
* Search podcasts
* Subscribe to podcast
* Play episode
* Download and offline play
* No playlists, no autoplay next episode
* No accounts, no sync
