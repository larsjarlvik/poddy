# Poddy
A simple podcast player in the jungle of over complicated podcast players.

*Note!* Work in progress, not in a working state

## Planned features
* Search podcasts
* Subscribe to podcast
* Play episode
* Download and offline play
* No playlists, no autoplay next episode
* No accounts, no sync
